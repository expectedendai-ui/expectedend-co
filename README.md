# expectedend.co

_An infinite 3D canvas of Denzel Rigaud — builder, believer, founder of MyBibleLens._

Drag to pan · scroll or pinch to zoom · keyboard (WASD / arrows) to navigate.

## Live

🔗 https://expectedend.co

## Stack

- React 19 + TypeScript
- React Three Fiber 9 + drei 10 (3D scene + media planes)
- Three.js 0.182
- Vite 7
- Brand palette: parchment / deep brown / gold / ember

## Develop

```bash
npm install
npm run dev
```

## Photos

Photos live in `public/artworks/` (24 images of mine, named `01.jpg` … `23.jpg`).
The accompanying metadata is at `src/artworks/manifest.json`.
